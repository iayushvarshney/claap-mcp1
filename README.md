# Claap MCP for Claude Desktop for Probabl

Connect Claap to Claude Desktop in one command.

## Install

### Step 1 — Get your Claap API Key
1. Go to [app.claap.io](https://app.claap.io)
2. Click avatar → Settings → API
3. Generate and copy your API key

### Step 2 — Run this in Terminal
```bash
curl -s https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/claap-mcp/main/install.sh | bash
```

### Step 3 — Restart Claude Desktop
Press `Cmd + Q` and reopen the app.

## Done! 🎉
Look for the 🔨 hammer icon in Claude Desktop chat.

## Test it
Type in Claude Desktop:
> "List my recent Claap recordings"
> "Summarize my last Claap meeting and extract action items"

## Requirements
- Mac (Apple Silicon or Intel)
- Claude Desktop app
- Node.js (the script will tell you if you need to install it)
